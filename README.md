# btrfs-debian-install-scipt

This repo contains two bash scripts containing every command needed to install a debian base system with btrfs, grub integration and snapper. It can install with either gnome, KDE or xfce as a desktop environment. The Bash Scripts are based on this guide: https://sysguides.com/install-debian-13-with-btrfs

This script is modifies the original guide in a way, that it is nvme-only and the Desktop Environment not being hardcoded.
